# Client-React-pour-API-REST 

## Description
Ce projet est un **TP d’architecture microservices** qui met en œuvre :
- Un **backend Spring Boot** exposant une API REST pour gérer des comptes bancaires  avec JPA et H2.
- Un **frontend React** consommant cette API pour afficher la liste des comptes et ajouter de nouveaux comptes via un formulaire.
L’objectif est de pratiquer l’intégration d’un frontend React avec un backend RESTful.
